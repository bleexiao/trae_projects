# Practice07: 文章初始化与生成系统 - 工作流程文档

## 项目概述

Practice07 是一个基于 init-article 技能的文章生成系统，能够根据用户需求自动生成符合规范的读书心得文章。

### 详细步骤说明

#### 步骤1: 读取 SKILL.md

```python
skill_path = os.path.join(os.path.dirname(os.path.abspath(__file__)), "init-article", "SKILL.md")
with open(skill_path, 'r', encoding='utf-8') as f:
    skill_content = f.read()
```

**SKILL.md 内容结构：**
- 文件头（Front Matter）：name 和 description
- 工作流程说明：四个规则文件的定位和作用
- 示例参考：指向 assets 目录下的示例文件

#### 步骤2: 构建提示词

```python
article_prompt = f"""请仔细阅读以下SKILL.md文件，理解其中的工作流程。

【SKILL.md内容】
{skill_content}

【用户需求】
{user_request}

请按照SKILL.md中描述的工作流程：
1. 理解四个规则文件的定位
2. 根据用户需求，在内部构建这四个规则
3. 严格按照这四个规则撰写文章

要求：
- 直接输出文章内容（txt格式）
- 不要输出规则文件内容
- 不要输出任何说明文字
- 只输出用户需要的文章正文"""
```

#### 步骤3: 调用LLM API

```python
messages = [
    {"role": "system", "content": "你是专业的文案撰写师..."},
    {"role": "user", "content": article_prompt}
]
response = send_request(messages)
```

**请求配置：**
- URL: `{BASE_URL}/v1/chat/completions`
- 模型: `{MODEL_NAME}`
- 认证: `Bearer {API_TOKEN}`
- 编码: UTF-8

#### 步骤4: 保存文章

```python
article_path = os.path.join(os.path.dirname(os.path.abspath(__file__)), "读书心得.txt")
with open(article_path, 'w', encoding='utf-8') as f:
    f.write(article)
```

## 与 Trae 自带 Skill 的对比

### 是否使用了 Trae 自带的 Skill 技能？

**答案：没有使用**

| 对比项 | Trae自带Skill | Practice07实现 |
|--------|---------------|----------------|
| **调用方式** | 通过 `Skill` 工具调用 | 手动读取SKILL.md文件 |
| **技能加载** | 自动加载并注入上下文 | 用Python代码手动读取 |
| **规则生成** | AI自动在内部处理 | 手动构建提示词 |
| **工具调用** | 支持多轮工具链调用 | 单次LLM调用 |
| **灵活性** | 受IDE框架限制 | 完全自定义 |

### 具体差异

**Trae自带Skill的工作方式：**
```
用户请求 → Skill工具 → 自动加载SKILL.md → AI理解并执行 → 输出结果
```

**Practice07的工作方式：**
```
用户请求 → Python代码读取SKILL.md → 手动构建提示词 → 调用LLM API → 保存文件
```

### 优势与劣势

**Practice07的优势：**
1. 完全自主控制流程
2. 不依赖IDE环境
3. 可独立运行和部署
4. 便于理解和调试

**Practice07的劣势：**
1. 需要手动维护所有逻辑
2. 缺少IDE的自动化工具支持
3. 无法利用Trae的内置功能

## 使用方法

1. 配置 `.env` 文件：
   ```
   BASE_URL=http://localhost:1234/v1
   MODEL_NAME=qwen3.5-9b
   API_TOKEN=your_token
   ```

2. 运行程序：
   ```bash
   cd practice07
   python chat_client.py
   ```

3. 选择操作：
   - 选择 1：一键生成读书心得
   - 选择 2：退出程序

4. 输入需求：
   ```
   我需要写一个读书心得，三千字，一本书是C语言程序设计，一本是JavaScript全栈开发
   ```

5. 等待生成完成，文章保存在 `practice07/读书心得.txt`
